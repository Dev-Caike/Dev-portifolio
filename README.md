## Como Contribuir
1. Crie uma branch: `git checkout -b minha-feature`
2. Commit suas mudanças: `git commit -m 'feat: descricao'`
3. Envie para o GitHub: `git push origin minha-feature`
4. Abra um Pull Request para a branch develop.
